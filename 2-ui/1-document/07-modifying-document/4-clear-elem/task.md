# Membersihkan elemen

Buatlah fungsi `clear(elem)` yang menghapus semua konten dari elemen tersebut.

```html run height=60
<ol id="elem">
  <li>Hello</li>
  <li>World</li>
</ol>

<script>
  function clear(elem) {
    /* kode anda */
  }

  clear(elem); // menghapus isi daftar
</script>
```
