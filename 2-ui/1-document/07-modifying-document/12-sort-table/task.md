# Mengurutkan Tabel

Berikut adalah sebuah tabel:

```html run
<table>
  <thead>
    <tr>
      <th>Nama</th>
      <th>Nama Belakang</th>
      <th>Usia</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>John</td>
      <td>Smith</td>
      <td>10</td>
    </tr>
    <tr>
      <td>Pete</td>
      <td>Brown</td>
      <td>15</td>
    </tr>
    <tr>
      <td>Ann</td>
      <td>Lee</td>
      <td>5</td>
    </tr>
    <tr>
      <td>...</td>
      <td>...</td>
      <td>...</td>
    </tr>
  </tbody>
</table>
```

Mungkin ada lebih banyak baris di dalamnya.

Tulislah kode untuk mengurutkannya berdasarkan kolom `"Nama"`.
